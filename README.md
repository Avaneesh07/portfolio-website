## Features

### Hero Section
- Animated introduction
- Tagline: "Technical Artist | VFX & AI-Driven Storyteller"
- Call-to-action buttons (View Work, Contact Me)

### Projects Showcase
- **Mars Outdoor Ads**: LED truck campaigns, vinyl wraps, holograms
- **VFX & 3D Work**: Camera assembly, physics sims, abstract visuals
- **AI Storytelling**: Spider-Verse ad, AI avatars, 100 Days series

### About Section
- Bio and background
- Skills: Blender, After Effects, AI tools, React, JavaScript
- Links to GitHub, LinkedIn, Instagram

### Contact Section
- Email form
- Social media links
- Download resume button

### AI Chatbot Integration
- Google AI Studio-powered chatbot
- Answers questions about my work
- Context-aware responses

## Tech Stack

- **Frontend**: React, JavaScript, HTML, CSS
- **Deployment**: Netlify
- **Version Control**: Git, GitHub
- **AI Integration**: Google AI Studio API
- **Styling**: CSS3, Responsive Design
